  • Balises de structure :

    • Les pages Web ont deux conteneurs de base: la <head></head> et le <body></body>:
    La balise <head><head>, qui contient des informations ou des métadonnées (Metadata) sur la page Web,
    comme un titre <title></title> pour la page, des javascripts, des styles (CSS)des métadonnées (Metadata),
    etc.Tout ce qui se trouve à l'intérieur de la balise <head></head> ne s'affiche pas sur la page Web elle-même.

    La balise <body></body>, le contenu qui s'affiche réellement dans un navigateur par exemple ( Internet explorer , Mozilla Firefox , Google Chrome , etc ) .

    La balise <title></title> définit le titre du document affiché dans la barre de titre d'un navigateur ou l'onglet d'une page.
    Il ne contient que du texte; les balises de l'élément sont ignorées .


    La balise <header><header>élément représente un conteneur de contenu d'introduction ou un ensemble de liens de navigation.

    La balise <main><main> représente le contenu dominant du <body> d'un document. La zone de contenu principale se compose de contenu qui est directement lié ou
    développe le sujet central d'un document ou la fonctionnalité centrale d'une application.

    La balise <footer><footer>balise définit un pied de page pour un document ou une section.

Liens et ancres

Liens

Ceci est un lien

Et en code:

Ceci est un <a href="https://www.w3schools.com/tags/tag_a.asp" target="_blank">lien</a>

Le <a> est la balise du lien hypertexte, le href est l'attribut le plus important qui amène sur une autre page.

Par défaut on clique sur un lien, il s'ouvrira dans la fenêtre actuelle. Pour changer cela il faut modifier l'attribut "target".

Il existe 9 attributs dont vous pouvez trouver la liste sur w3schools, mais voici une courte liste des attributs principaux

Attributs Valeurs Description
href URL sur laquelle est amené le visiteur
target
  • _blank
  • _parent
  • _self
  • _top
  • framename
 Spécifie où l'url sera ouverte
download filename Spécifie que la cible du lien sera downloadée quand on clique dessus

Ancres

Une ancre est un lien vers une partie de la page sur laquelle se trouve le visiteur. Ce lien vous amènera en haut de cette page.

Et en code:

Cette <a href="#liens" >ancre</a> vous amènera sur la partie sur les liens de cette page.

Pour faire une ancre il faut:

  • Créer l'ancre en elle-même: <a id="liens">Liens</a>
  • Faire le lien avec l'ancre: Ce <a href="#liens">lien</a> vous amènera(...)
