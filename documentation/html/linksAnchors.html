Liens et ancres

Liens

Ceci est un lien

Et en code:

Ceci est un <a href="https://www.w3schools.com/tags/tag_a.asp" target="_blank">lien</a>

Le <a> est la balise du lien hypertexte, le href est l'attribut le plus important qui amène sur une autre page.

Par défaut on clique sur un lien, il s'ouvrira dans la fenêtre actuelle. Pour changer cela il faut modifier l'attribut "target".

Il existe 9 attributs dont vous pouvez trouver la liste sur w3schools, mais voici une courte liste des attributs principaux :

Attributs Valeurs Description
href URL sur laquelle est amené le visiteur
target
  • _blank
  • _parent
  • _self
  • _top
  • framename
 Spécifie où l'url sera ouverte
download filename Spécifie que la cible du lien sera downloadée quand on clique dessus

Ancres

Une ancre est un lien vers une partie de la page sur laquelle se trouve le visiteur. Ce lien vous amènera en haut de cette page.

Et en code:

Cette <a href="#liens" >ancre</a> vous amènera sur la partie sur les liens de cette page.

Pour faire une ancre il faut:

  • Créer l'ancre en elle-même: <a id="liens">Liens</a>
  • Faire le lien avec l'ancre: Ce <a href="#liens">lien</a> vous amènera(...)
