Les listes

1 - Introduction aux listes HTML

HTML offre plusieurs moyens d’établir des listes d'informations. Toutes les listes doivent contenir un élément de liste ou plus. Celles-ci peuvent présenter des informations non-ordonnées, ordonnées ou des définitions. Nous allons voir ici comment les construire.

1.1 - Les listes ordonnées et non-ordonnées

Les listes non-ordonnées n'ont pas de notion d'ordre et on peut mélanger leurs éléments sans que cela modifie leur sens. Elles apparaissent parfois à des endroits inattendus: les barres de navigation sont par exemple des listes non-ordonnées auxquelles on a appliqué quelques règles CSS pour la mise en forme. Lorsque les éléments de la liste doivent être lus ou réalisés dans un ordre précis, on utilisera la liste ordonnée.

Syntaxe HTML :

Toute la liste sera placée dans les balises <ol> pour les listes ordonnées ou <ul> pour les listes non-ordonnées. Les balises fermentes </ol> et </ul> sont obligatoires. Chacun des éléments de la liste sera placé à l'intérieur d'un élément <li>. La balise fermante </li> est optionnelle. Voici un exemple qui contient les deux types de liste :

<h1>Guacamole rapide à faire </h1> <h2>Ingrédients</h2> <ul> <li>2 avocats </li> <li>le jus d'un citron</li> <li>¼ de concombre, coupé grossièrement</li> <li>1 petite tomate, coupée</li> </ul> <h2>Instructions</h2> <ol> <li>Écrasez délicatement les avocats avec une fourchette</li> <li>Placez la purée obtenue dans un plat et arrosez avec le jus de citron</li> <li>Mélangez pour que le jus de citron empêche la purée d'avocat de noircir</li> <li>Mélangez la tomate et le concombre coupés</li> <li>Gardez au frais et servir rapidement avec des tortillas</li> </ol>

Guacamole rapide à faire

Ingrédients

  • 2 avocats
  • le jus d'un citron
  • ¼ de concombre, coupé grossièrement
  • 1 petite tomate, coupée

Instructions

  1. Écrasez délicatement les avocats avec une fourchette
  2. Placez la purée obtenue dans un plat et arrosez avec le jus de citron
  3. Mélangez pour que le jus de citron empêche la purée d'avocat de noircir
  4. Mélangez la tomate et le concombre coupés
  5. Gardez au frais et servir rapidement avec des tortillas

Par défaut, le navigateur utilisera des chiffres classiques pour les listes ordonnées et des puces (rondes) pour les listes non-ordonnées. Nous verrons par la suite comment modifier ces paramètres.

1.2 - Les listes de définition

La liste de définitions consiste généralement en une succession de couples terme/définition (même si les listes peuvent avoir d'autres usages). Ainsi, on pourrait utiliser une liste de définitions pour la publicité d'un produit.

<dl> <dt>Titre de la définition</dt> <dd>La définition en elle même</dd> <dt>Titre de la 2 eme définition</dt> <dd>La définition 2eme en elle même</dd> </dl>

1.3 - Les listes imbriquées

On voit souvent plusieurs listes placées les unes dans les autres. Autrement dit, on peut imbriquer des listes (un élément d'une liste sera une liste entière) :

<ul> <li>Un composant</li> <li>Un processus :<> <ol> <li>Étape 1</li> <li>Étape 2</li> <li>Étape 3</li> </ol> </li> <li>Un composant</li> </ul>
  • Un composant
  • Un processus :
    1. Étape 1
    2. Étape 2
    3. Étape 3
  • Un composant

Les menus de navigation utilisent parfois des listes imbriquées pour afficher les sous-catégories. Pour étudier un exemple réel, vous pouvez inspecter le code HTML du menu MDN.

2 - Mise en forme d’une liste HTML

2.1 - Sans CSS

Les agents utilisateurs visuels place généralement en retrait les listes imbriquées en fonction du niveau d'imbrication courant. La présentation exacte des trois listes dépend de l'agent utilisateur.

Les differents attributs :

- type = "indication-de-style"

Pour les deux éléments OL et UL, cet attribut fixe le style d'un item de liste. Pour l'élément UL, les valeurs possibles de l'attribut type sont "disc", "square", "circle" ou « none ».

La valeur par défaut dépend de la profondeur d'imbrication de la liste courante. Ces valeurs sont insensibles à la casse.
Exemple de puce carré avec l’element UL :

<ul type= "square"> <li>Je suis le 1e item avec une puce de forme carrée</li> <li>Je suis le 2e item avec une puce de forme carrée</li> <li>Je suis le 3e item avec une puce de forme carrée</li> </ul>
  • Je suis le 1er item avec une puce de forme carrée
  • Je suis le 2e item avec une puce de forme carrée
  • Je suis le 3e item avec une puce de forme carrée

Pour l'élément OL, les valeurs possibles de l'attribut type sont résumées ci-dessous, suivant le style de numerotation souhaité (les valeurs sont sensibles à la casse) :

  • style = "1" restitue des chiffres arabes 1, 2, 3, ...
  • style = "a" restitue des lettres en minuscules a, b, c, ...
  • style = "A" restitue des lettres en majuscules A, B, C, ...
  • style = "i" restitue des chiffres romains minuscules i, ii, iii, ...
  • style = "I" restitue des chiffres romains majuscules I, II, III, ...

Exemple de numerotation en chiffres romains minuscules avec l’element OL :

<ol type= "i"> <li>Je suis le 1er de la liste avec une numérotation i </li> <li>Je suis le 2er de la liste avec une numérotation ii </li> <li>Je suis le 3er de la liste avec une numérotation iii </li> </ol>
  1. Je suis le 1er de la liste avec une numérotation i
  2. Je suis le 2er de la liste avec une numérotation ii
  3. Je suis le 3er de la liste avec une numérotation iii

- start = "nombre"

Seulement pour l'élément OL. Cet attribut spécifie le numéro initial du premier item dans une liste ordonnée. Le numéro initial par défaut est "1". Tandis que la valeur de cet attribut est un entier, le label correspondant peut ne pas être numérique. Ainsi, si start = "3" et type = "A", le 1er item de la liste commencera a "C".

<ol start="3" type="A"> <li>Je suis premier</li> <li>Je suis deuxième</li> <li>Je suis troisième</li> </ol>
  1. Je suis premier
  2. Je suis deuxième
  3. Je suis troisième

- reversed

Pour décompter plutôt que compter (par exemple 5-4-3-2-1), on pourra utiliser l'attribut reversed de l'élément OL :

Exemple Syntaxe HTML : <ol reversed>

- value = "nombre"

Seulement pour l'élément LI. Cet attribut fixe le numéro de l'item de liste courant. Tandis que la valeur de cet attribut est un entier, le label correspondant peut être non-numérique. (voir l'attribut start).

Exemple Syntaxe HTML : <li value="4">;. Les éléments suivants de la liste seront également renumérotés : 4, 5, 6, etc.).

- compact

Quand il est spécifié, cet attribut booléen indique aux agents utilisateurs de restituer la liste de manière plus compacte. L'interprétation de cet attribut dépend de l'agent utilisateur.

Nous décourageons les auteurs d'utiliser les listes simplement comme un moyen pour placer le texte en retrait. C'est une affaire de style qui est du ressort des feuilles de style.

2.2 - Avec CSS

Les attributs HTML permettent de modifier le contenu d'une liste. Pour ajuster l'aspect cosmétique d'une liste, il est vivement recommandé d’utiliser CSS.
Trois solutions existent pour appliquer des modifications :

  • directement dans le fichier CSS de votre thème
  • dans l’en-tête de votre page HTML
  • directement sur la balise

Pour plus de simplicité, je ferais ici l’application directement sur la balise.

- Modifier le style de puces avec list-style-type

Pour modifier le style de puce ou le style de numérotation, on pourra utiliser la propriété CSS list-style-type. Pour cela, on ajoute l’argument "style" et on utilise la propriété "list-style-type" dans la balise HTML.

Mise en forme liste non-ordonnée

Pour l’element UL, le style par defaut est "disc". Ci-dessous un exemple de code pour chaque puce.

<li>Exemple liste à puce HTML – cercle 1</li> <li> Exemple liste à puce HTML – cercle 2</li> </ul>
  • Exemple liste à puce HTML - cercle 1
  • Exemple liste à puce HTML - cercle 2
<ul style= "list-style-type: square"> <li>Exemple liste à puce HTML – carré 1</li> <li> Exemple liste à puce HTML – carré 2</li> </ul>
  • Exemple liste à puce HTML - carré 1
  • Exemple liste à puce HTML - carré 2
<ul style= "list-style-type: none"> <li>Exemple liste à puce HTML – sans puce 1</li> <li> Exemple liste à puce HTML – sans puce 2</li> </ul>
  • Exemple liste à puce HTML – sans puce 1
  • Exemple liste à puce HTML – sans puce 2

CSS peut permettre d'obtenir des effets très complexes (comme un système de comptage personnalisé). Si vous souhaitez apprendre CSS, vous pouvez parcourir le Guide CSS et voir comment appliquer des règles CSS à une page web.

Prenons un rapide exemple. Voici une liste HTML non-ordonnée :

  • J'ai un point
  • J'ai un cercle
  • J'ai aussi un cercle

Ces règles CSS permettront que la puce des éléments soit un cercle, sauf pour le premier (où ce sera un point) :

ul {
list-style-type: circle;
}
li:first-child {
list-style-type: disc;
}

Cela donnera :

  • J'ai un point
  • J'ai un cercle
  • J'ai aussi un cercle

Mise en forme liste ordonnee

Pour l’élément OL, le meme principe s’applique pour modifier le style de numerotation. Voici les principales possibilites :

  • decimal => par défaut – chiffres décimaux
  • decimal-leading-zero => les chiffres décimaux avec un zéro devant (01, 02, 03…)
  • upper-roman => chiffres romains en majuscules
  • lower-roman => chiffres romains en minuscules
  • upper-alpha => alphabet en majuscules
  • lower-alpha => alphabet en minuscules
  • lower-greek => chiffres grecs

Exemple avec l’alphabet en majuscule:

<ol style="list-style-type:upper-alpha"> <li>Exemple liste numerotation lettre majuscule</li> <li> Exemple liste numerotation lettre majuscule</li> </ol>
  1. Exemple liste numerotation lettre majuscule
  2. Exemple liste numerotation lettre majuscule
Mise en forme liste de definition

Concernant les listes de définition, il n’y a pas de paramètre par défaut permettant de faire des modifications. Ici, il s’agit plutôt de faire ressortir les informations et différencier le mot ou l’expression de la définition de cette dernière.
La balise HTML contenant le mot ou l’expression étant <dt>, on peut la faire ressortir en appliquant du CSS à la volée comme nous l’avons vu précédemment. Ci-dessous, un exemple qui permettra une meilleure différenciation avec la définition.

<p>La syntaxe :</p> <dl> <dt> style= "color:white; background-color:black; font-weight:bold;">Titre de la définition</dt> <dd>La définition en elle même</dd> <dt>Titre de la 2eme définition</dt> <dd>La définition 2eme en elle-même<dd> </dl>

- Utiliser une image à la place d’une puce avec list-style-image

Pour utiliser une image donnée comme puce, on pourra utiliser la propriété CSS list-style-image. Attention toutefois, les images sont plus lourdes (donc plus lentes) et moins adaptatives que le texte. Ci-dessous un exemple de personnalisation d’une liste à puce, j’ai utilisé l’image favicon du site internet, c’est pour vous montrer qu’on peut utiliser n’importe quelle image. Il faudra simplement faire attention à l’aspect esthétique, en l’occurrence ici, ce n’est pas top. Mais c’est simplement pour la démonstration.
Exemple de puce avec le logo HTML5 :

<ul style="list-style-image:url('img/alpha/HTML5_logo_reduced.png');"> </li>Exemple liste à puce HTML logo</li> </li>Exemple liste à puce HTML logo</li> </ul>
