Open Graph

Open Graph est un protocole créé par Facebook en 2010 puis confiée à l'Open Web Foundation.

Open Graph permet de transformer un site en objet riche dans un graphe social et d'optimiser le contenu d'une page web afin que les utilisateurs puissent la partager sur les réseaux sociaux Facebook, Twitter, LinkedIn...

Anecdote: Lorsque vous mettez un lien dans facebook (une vidéo Youtube par ex.) et que cela charge directement la miniature de la video, le petit texte descriptif ect pour votre prochaine publication, ce n'est pas pour rien ;)

Pour mentionner de l’open Graph, on utilisera la balise <meta> et plusieurs attributs spéciaux ( property=”og:...”) pour transformer le site en média riche pour les réseaux sociaux.

Les métadonnées classiques

Quelques exemples :

property="og:title" : Le titre de l’objet riche

property="og:type : Le type de l’objet riche

property="og:url” : L’URL de l’objet riche

property="og:image” : L’URL de l’objet riche

    <meta property="og:title" content="The Rock" />
    <meta property="og:type" content="video.movie" />
    <meta property="og:url" content="http://www.imdb.com/title/tt0117500/" />
    <meta property="og:image" content="http://ia.media-imdb.com/images/rock.jpg" />

Il existe aussi des propriétés structurées, des tableaux, des types d’objets etc..

Plus d’infos sur l’Open Graph : https://opengraphprotocol.org/
