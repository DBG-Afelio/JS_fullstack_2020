Ecrire un programme qui demande à l’utilisateur d’entrer un texte.
Le programme affichera le texte en ayant inversé les mots.

Remarque :
Les mots sont des sous-chaines ne contenant pas d’espace.


Debut ex5-7
    VARIABLES LOCALES :
        var txt :T <- ""
        var final :T <- ""
        var mot :T <- ""
        var i :N <- 0
         var firstWord :B <- FAUX
    FIN VARIABLES LOCALES

    Afficher "Entrez le texte"
    Saisir txt
    
    i <- 0
    mot <- ""
    final <- ""
    firstWord <- FAUX
    POUR i ALLANT DE 0 A longueur(txt)-1
        SI (carac(txt, i)) = ' ')
            ALORS
                SI (firstWord)
                    ALORS                         
                        mot <- sousChaine(txt, pos, i)
                        final <- mot (+) final 
                    SINON
                        firstWord <- VRAI
                FIN SI
                pos <- i+1
        FIN SI
    i SUIVANT

    mot <- sousChaine(txt, pos, longueur(txt)-1)
    final <- mot (+) final

    Afficher final
 
Fin ex5-7
