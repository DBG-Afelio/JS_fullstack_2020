Les balises multimédia

Les balises multimédia permettent de lire du contenu audio et vidéo sur le navigateur et une d’elle, donne le droit à l’utilisateur de dessiner des motifs et des formes sur une zone spécifique du navigateur.

Liste des balises media :

<Audio> : Cette balise permet l’affichage et la prise en charge de contenu audio, sous différents formats.

Pour spécifier la piste audio, on peut utiliser l’attribut src ou la balise source. L’attribut controls met à disposition de l’utilisateur une interface pour la lecture du média.

Quelques formats audio : mp3, ogg, wav, flac...

Quelques attributs utiles : autoplay, loop, muted, preload

<Video> : Permet de lire du contenu vidéo sous plusieurs formats. Comme la balise audio, l’attribut src et la balise source.

De même, L’attribut controls peut être spécifié pour disposer de l’interface de contrôle.

Formats audio : mp4, avi, mov, rm, mkv, divx...

Attributs utiles : autoplay, loop, muted, preload

<Source> (balise orpheline) : La balise source sert à définir une ou plusieurs ressources média pour sa lecture (s’accompagne avec la balise audio ou vidéo)

Quelques attributs utiles : media, sizes, src, srcset, type

<Embed> (balise orpheline) : La balise embed permet d'intégrer un média ou une application sur la page HTML

Quelques attributs qui l’accompagne (conseillé de toutes les définir et spécifier): height, src, type, width

<Picture>: Cette balise agit comme les balises audio et vidéo et mais pour les images

Quelques attributs utiles : media, sizes, src, srcset, type

<img> (balise orpheline) : Affiche une image.

Petite précision CSS : Les images sont définies en mode display: inline par défaut

Format d’images prise en charge : jpeg, png, svg, tiff, gif, webP, bmp…

Attributs pris en charge : alt, crossorigin, sync, async, ismap, size, src, srcset, width..

<object> permet d'insérer une ressource de type media, application ou plugin.

Petite précision : Les nouveaux navigateurs ne sont plus compatible avec Adobe Flash, ActiveX, les applets et plugins Java

Attributs pris en charge : data, form, width, height, type, name...

<param> sert à définir les paramètres qui peuvent être employés dans un élément qui a pour balise object.

Attributs : value, name

<track> permet d’ajouter des sous-titres dans un média de type audio et vidéo.

Attributs : default, kind (subtitles, captions, descriptions, chapters, metadata), label, src, srclang

<canvas>

Cette balise permet de créer une zone sur le navigateur sur lequel l’utilisateur peut injecter du code javascript (via l’API Canvas) pour créer des animations, faire du dessin SVG et des graphes, voir même de la 3D (via l’API webGL).

<svg>

La balise SVG permet d'insérer du code SVG de définir des coordonnées en 2 dimensions pour reproduire des formes (polygon, rectangle, cercle, sphère...) , des dessins et même des animations.

Les balises de description de document

Les balises de description de document servent à ajouter des informations à une page html pour l’enrichir, faciliter la recherche et l’indexation par les robots de moteur de recherche.

Liste des balises de description :

<meta> Les balises META servent à placer des métadonnées dans une page HTML. On placera ces informations dans l’élément head, et elles ne seront pas affichées sur la page.

Exemple d’utilisation de balises meta :

- Définir le jeu de caractère d’un document (meta charset=”UTF-8”)

- (Sur mobile) Bloquer le zoom d’une page pour garder le contrôle en cas de zoom de l’utilisateur (meta name="viewport" content="width=device-width, initial-scale=1.0")

Attributs : charset, content, refresh, name, color-scheme

<link> La balise link sert à définir un lien vers une ressource externe (une feuille de style, un fichier de script javascript, un média...)

Attributs : href, media, rel, sizes, title, type....

<title>La balise title définit le nom du document HTML(Celui-ci sera repris sur le titre de l’onglet actuel sur votre navigateur)

SEO: Le titre de la page est un des premiers éléments scanne par les robots d’indexation.

<script> Permet d'intégrer du code javascript au pages HTML :

- A l'intérieur des balises (<script></script>)

- A l’aide d’un fichier externe ( <script src=”main.js”></script> )

Attributs: async, defer, src, type...
