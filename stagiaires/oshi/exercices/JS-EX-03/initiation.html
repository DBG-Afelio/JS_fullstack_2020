Initiation à la programmation Javascript

Il s'agit ici d'un premier exercice pour initier l'apprentissage de la programmation Javascript dans un browser.
Pour chaque exemple, il faut trouver le bon évènement et la bonne propriété DOM à modifier.

Un champ qui se vide lorsque l'on entre dedans.
Event : focus, blur
Property : value

Un champ qui se vide lorsque l'on entre dedans et qui remet la valeur initiale si le champ est vide lorsqu'on le quitte.
Event : focus, blur
Property : value, defaultValue

Un champ dont la taille augmente lorsque l'on appuye sur une touche et qu'on est dedans.
Event : keydown
Property : size

Un champ qui affiche les caractères quand la case est cochée, les masque sinon.
Event : click
Property : checked, type

Un champ qui affiche les caractères quand on maintient la souris enfoncé sur le carré, les masque sinon.
Event : mousedown, mouseup
Property : checked, type

Un champ qui jete le focus quand essaye de lui donner.
Event : focus
Method : blur()

Un champ qui jete le focus quand essaye de lui donner et que la case à cochée n'est pas activée.
Si le champ est rempli et que l'on décoche la case, on vide le champ. Si on coche la case, on amène le focus dans le champ, et on place la valeur 0;

Un champ non modifiable qui s'incrémente ou se décrémente selon l'option sélectionnée
Event : click
Property : checked, value


Le bouton "sauve" place la valeur actuelle du champ comme valeur par défaut. Le bouton "reset" fait son boulot...
Event : click
Property : value, defaultValue

Place la valeur et le texte de l'option sélectionnée à l'intérieur des champs correspondants.
Event : click
Property : options(array), selectedOption, text, value



Place les valeurs et les textes des options sélectionnées à l'intérieur des champs correspondants .



Permutte les valeurs des différents champs dans le sens de la flèche (en boucle).

Faire basculer les items sélectionnés entre les deux listes.
